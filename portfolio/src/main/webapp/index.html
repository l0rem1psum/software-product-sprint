My Projects

I am very passionate about programming and here are a list of things I have worked on in the past:

Building Microservices for U-Wave
- U-Wave is an NTU's all-in-one campus app. The microservices are built with Spring Cloud and based on many middlewares such as Redis and RabbitMQ. Apps are dockerized and deployed with Kubernetes on GCP.
Dockerized Firebase SCrypt
- A dockerized version of Firebase's implementation of SCrypt, proxied by Python Flask.
MySQL failover with Keepalived
- A demo of achieving highly available MySQL by using Keepalived for failover.
OTPy
- A Python package implementing RFC 6238 for One-Time Password (OTP).
Circuit Master
- An Android game written in C# to teach players how logic gates work.
Github Repositories Searcher
- A simple Vue.js app for searching Github users and view README files of their projects.
